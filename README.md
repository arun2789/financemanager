# Rupee Nest

A clean India-first personal finance web app for individuals. It helps track expenses, connected money sources, budgets, bills, and monthly savings from one uncluttered dashboard.

## Features

- INR-first summary cards for balance, monthly spend, savings, and bills
- Expense capture with merchant, category, source, date, type, and notes
- Source cards for bank accounts, credit cards, UPI, wallets, and CSV imports
- Monthly category budgets with progress bars
- Upcoming bills and EMI tracking
- Local browser storage, so the demo data and new expenses persist in your browser

## Run

```bash
npm start
```

Open:

```text
http://localhost:5173
```

The app is static HTML, CSS, and JavaScript in `src/`.
